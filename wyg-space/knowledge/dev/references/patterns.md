# DEV 模式 - 开发实现模式库

## 模式1: 方案→实现映射

| 方案元素 | 实现动作 |
|---------|---------|
| 功能点 | 逐个实现，不遗漏 |
| 接口定义 | 严格按接口编码 |
| 异常处理 | 每个异常场景都有处理 |
| 备选方案 | 实现主方案+降级逻辑 |

## 模式2: 错误记忆驱动修复

```
1. 读取错误记忆
2. 理解错误原因和教训
3. 定位需要修改的部分
4. 实现修复（避免重复错误）
5. 不删除错误记忆
```

## 模式3: 渐进式实现

```
骨架 → 核心功能 → 边界处理 → 优化完善
```

适用场景：
- 方案较复杂
- 需要逐步验证
- 有时间压力
